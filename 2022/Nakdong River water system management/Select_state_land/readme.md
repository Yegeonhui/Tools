# Select_river_valley
> 지적도에서 국유지인 부분만 shapefile, geoJSON파일로 저장

## How to use?
1. 지목 중 국유지인 부분만 선택하고
2. shapefile, geoJSON으로 저장

## install Library
```
conda install geopandas
```
